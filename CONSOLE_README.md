# 体重管理控制台系统

这是一个完整的体重管理平台后台控制台系统，基于 React + React Router + Tailwind CSS + Recharts 构建。

## 功能模块

### 1. 首页（Dashboard）
- 核心业务指标卡片（代餐业务、设备业务、用户活跃）
- 近7日业务趋势图（订单量、设备绑定量、用户活跃量）
- 待办事项快速入口
- 数据异常预警提示

### 2. 数据埋点

#### 2.1 数据概览
- 用户活跃指标（DAU、MAU、留存率）
- 核心流程转化率
- 业务转化数据
- 异常监控统计
- 核心漏斗简览
- 活跃趋势图表

#### 2.2 数据分析
- 漏斗分析（导诊流程、设备绑定、代餐转化、打卡流程）
- 用户行为分析（留存趋势、页面访问排行）
- 用户分布（来源、设备绑定、购买情况）
- 异常监控详情

### 3. 代餐服务

#### 3.1 服务包维护
- 服务包列表查询与筛选
- 新增/编辑服务包（基础信息、产品信息、素材管理、审核提交）
- 批量上下架操作
- 预览功能

#### 3.2 价格维护
- 价格列表管理
- 原价、售价、活动价设置
- 库存管理与预警
- 限购设置
- 价格变更记录查询
- 批量调价工具

### 4. 订单管理

#### 4.1 订单记录
- 订单列表查询（多维度筛选）
- 发货操作（单个/批量）
- 订单详情查看
- 售后处理

#### 4.2 发货记录
- 发货记录查询
- 物流轨迹查看
- 发货数据统计
- 超时未签收提醒

### 5. 用户管理

#### 5.1 用户列表
- 用户信息查询（已脱敏）
- 用户详情（基础信息、健康档案、订单记录、设备绑定、行为记录）
- 多维度筛选

#### 5.2 角色管理
- 预设角色（超级管理员、运营人员、客服人员、数据分析师）
- 自定义角色创建
- 权限配置（菜单级别 + 操作级别）

#### 5.3 菜单管理
- 菜单树结构管理
- 菜单类型（目录、页面、按钮）
- 启用/禁用控制
- 排序设置

### 6. 系统设置

#### 6.1 操作日志
- 全后台操作记录查询
- 操作详情查看
- 多维度筛选
- 日志导出

#### 6.2 消息模板
- 小程序订阅消息模板管理
- 内部通知模板管理
- 模板编辑与预览
- 变量替换支持

## 技术特点

1. **完整的路由系统** - 使用 React Router Data Mode 模式
2. **响应式设计** - 适配桌面端和平板设备
3. **数据可视化** - 使用 Recharts 展示图表
4. **组件化开发** - 清晰的组件结构和代码组织
5. **权限控制** - 角色权限管理和菜单权限控制
6. **数据脱敏** - 用户信息展示遵循隐私保护原则
7. **操作留痕** - 全程操作日志记录

## 数据安全

- 用户信息脱敏显示（昵称、ID、地址等）
- 操作日志全程记录
- 角色权限严格控制
- 最小权限原则

## 开始使用

系统默认以超级管理员身份登录，可访问所有功能模块。不同角色看到的菜单项和操作权限不同。

### 默认账号
- 姓名：张三
- 角色：超级管理员
- 权限：全模块全权限

## 注意事项

1. 所有数据为模拟数据，仅用于演示
2. 实际使用需要对接真实后端API
3. 图片上传功能需要实现文件上传接口
4. 消息推送需要对接微信小程序订阅消息API
